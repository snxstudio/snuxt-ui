A slide-out panel with header, body, and footer sections. Supports left, right, top, and bottom sides. Traps focus, locks scroll, and closes on Escape.

## Usage

```tsx
import {
  Drawer, DrawerHeader, DrawerTitle,
  DrawerDescription, DrawerBody, DrawerFooter,
} from '@snuxt-ui/react'
```

### Right Side (Default)

```tsx
const [open, setOpen] = useState(false)

<Button onClick={() => setOpen(true)}>Open Right</Button>
<Drawer open={open} onOpenChange={setOpen} side="right">
  <DrawerHeader>
    <DrawerTitle>Settings</DrawerTitle>
    <DrawerDescription>Adjust your preferences.</DrawerDescription>
  </DrawerHeader>
  <DrawerBody>
    <div className="flex flex-col gap-3">
      <Input placeholder="Name" />
      <Input placeholder="Email" />
    </div>
  </DrawerBody>
  <DrawerFooter>
    <Button onClick={() => setOpen(false)}>Save</Button>
  </DrawerFooter>
</Drawer>
```

### Left Side

```tsx
const [open, setOpen] = useState(false)

<Button onClick={() => setOpen(true)}>Open Left</Button>
<Drawer open={open} onOpenChange={setOpen} side="left">
  <DrawerHeader>
    <DrawerTitle>Navigation</DrawerTitle>
    <DrawerDescription>Browse sections.</DrawerDescription>
  </DrawerHeader>
  <DrawerBody>
    <nav className="flex flex-col gap-2">
      <a href="/dashboard">Dashboard</a>
      <a href="/settings">Settings</a>
      <a href="/profile">Profile</a>
    </nav>
  </DrawerBody>
</Drawer>
```

### Top Side

```tsx
<Drawer open={open} onOpenChange={setOpen} side="top">
  <DrawerHeader>
    <DrawerTitle>Notification</DrawerTitle>
    <DrawerDescription>You have new updates.</DrawerDescription>
  </DrawerHeader>
  <DrawerBody>
    <p>Check your inbox for details.</p>
  </DrawerBody>
</Drawer>
```

### Bottom Side

```tsx
<Drawer open={open} onOpenChange={setOpen} side="bottom">
  <DrawerHeader>
    <DrawerTitle>Quick Actions</DrawerTitle>
  </DrawerHeader>
  <DrawerBody>
    <div className="flex gap-2">
      <Button variant="outline">Share</Button>
      <Button variant="outline">Copy Link</Button>
      <Button variant="destructive">Delete</Button>
    </div>
  </DrawerBody>
</Drawer>
```

## API Reference

<PropsTable props={[
  { name: 'open', type: 'boolean', description: 'Whether the drawer is visible.' },

  { name: 'side', type: '"left" | "right" | "top" | "bottom"', default: '"right"', description: 'Which edge of the screen the drawer slides from.' },
  { name: 'children', type: 'ReactNode', description: 'Drawer sub-components.' },
]} />

### Sub-components

| Component | Description |
|---|---|
| `DrawerHeader` | Container for the title and description. |
| `DrawerTitle` | The heading of the drawer. Renders an `<h2>`. |
| `DrawerDescription` | Supplementary text below the title. Renders a `<p>`. |
| `DrawerBody` | Scrollable main content area of the drawer. |
| `DrawerFooter` | Container for action buttons, typically at the bottom. |
