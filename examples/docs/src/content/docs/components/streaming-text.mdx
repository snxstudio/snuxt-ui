Pure CSS animations for streaming AI text responses. Includes a blinking cursor, word/line fade-in animations, and a shimmer skeleton loader.

## Usage

```tsx
import {
  StreamingText, StreamingTextFadeIn, StreamingTextWord,
  StreamingTextLine, StreamingTextSkeleton, StreamingTextSkeletonLine,
} from '@snuxt-ui/react'
```

### Blinking Cursor

```tsx
<StreamingText>The response text appears here...</StreamingText>
```

### Fade-in Lines

```tsx
<StreamingText>
  <StreamingTextLine>First line.</StreamingTextLine>
  <StreamingTextLine style={{ animationDelay: '0.2s' }}>Second line.</StreamingTextLine>
</StreamingText>
```

### Skeleton Loader

```tsx
<StreamingTextSkeleton>
  <StreamingTextSkeletonLine />
  <StreamingTextSkeletonLine />
  <StreamingTextSkeletonLine />
</StreamingTextSkeleton>
```

## API Reference

### StreamingText

Wrapper with a blinking cursor `::after` pseudo-element.
