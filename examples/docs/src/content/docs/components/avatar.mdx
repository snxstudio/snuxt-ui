## Usage

```tsx
import { Avatar } from '@snuxt-ui/react'
```

### With Initials

```tsx
<Avatar fallback="SN" />
<Avatar fallback="UI" />
```

### With Image

```tsx
<Avatar src="/avatar.jpg" alt="User" fallback="U" />
```

### Fallback on Error

```tsx
// If the image fails to load, the fallback initials are shown
<Avatar src="/broken-link.jpg" alt="User" fallback="JD" />
```

### Group

```tsx
<div className="flex -space-x-2">
  <Avatar fallback="A" />
  <Avatar fallback="B" />
  <Avatar fallback="C" />
</div>
```
