Compact citation cards for displaying AI response sources. Renders in a 2-column grid with favicon, title, domain, and optional numbered index.

## Usage

```tsx
import { SourceCards, SourceCard } from '@snuxt-ui/react'
```

### Basic Grid

```tsx
<SourceCards>
  <SourceCard
    href="https://example.com"
    favicon="https://example.com/favicon.ico"
    title="Example Article"
    domain="example.com"
    index={1}
  />
</SourceCards>
```
