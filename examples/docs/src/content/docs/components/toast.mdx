A non-intrusive toast notification that appears at the edge of the screen and auto-dismisses. Supports default, success, and destructive variants.

## Usage

```tsx
import { useToast, Toaster } from '@snuxt-ui/react'
```

Make sure to render the `<Toaster />` component once at the root of your app:

```tsx
// App.tsx
import { Toaster } from '@snuxt-ui/react'

function App() {
  return (
    <>
      {/* your routes / content */}
      <Toaster />
    </>
  )
}
```

### Default Toast

```tsx
const { toast } = useToast()

toast({ title: 'Saved', description: 'Your changes have been saved.' })
```

### Success Toast

```tsx
toast.success('Success!', 'File uploaded successfully.')
```

### Destructive Toast

```tsx
toast.destructive('Error', 'Something went wrong. Please try again.')
```

### Custom Duration

```tsx
toast.custom({
  title: 'Reminder',
  description: 'This toast stays for 10 seconds.',
  duration: 10000,
})
```

### With Action Button

```tsx
toast.custom({
  title: 'File deleted',
  description: 'The file has been removed.',
  action: {
    label: 'Undo',
    onClick: () => restoreFile(),
  },
})
```

## API Reference

### `useToast()` hook

Returns an object with:

| Property | Type | Description |
|---|---|---|
| `toast` | `ToastManager` | Toast manager with `.default()`, `.success()`, `.destructive()`, and `.custom()` methods. |
| `toasts` | `Toast[]` | Current list of active toasts. |
| `dismiss` | `(id: string) => void` | Dismiss a specific toast by ID. |
