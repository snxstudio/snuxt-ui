A modal dialog that displays above the page with a backdrop overlay. Includes header, description, and footer sub-components. Traps focus, locks scroll, and closes on Escape.

## Usage

```tsx
import {
  Dialog, DialogHeader, DialogTitle,
  DialogDescription, DialogFooter,
} from '@snuxt-ui/react'
```

### Confirmation Dialog

```tsx
const [open, setOpen] = useState(false)

<Button onClick={() => setOpen(true)}>Delete Account</Button>
<Dialog open={open} onOpenChange={setOpen}>
  <DialogHeader>
    <DialogTitle>Are you sure?</DialogTitle>
    <DialogDescription>
      This action cannot be undone. This will permanently delete your
      account and remove your data from our servers.
    </DialogDescription>
  </DialogHeader>
  <DialogFooter>
    <Button variant="outline" onClick={() => setOpen(false)}>
      Cancel
    </Button>
    <Button variant="destructive" onClick={() => setOpen(false)}>
      Confirm
    </Button>
  </DialogFooter>
</Dialog>
```

### Alert Dialog

```tsx
const [open, setOpen] = useState(false)

<Button onClick={() => setOpen(true)}>Show Alert</Button>
<Dialog open={open} onOpenChange={setOpen}>
  <DialogHeader>
    <DialogTitle>Session Expired</DialogTitle>
    <DialogDescription>
      Your session has expired. Please log in again to continue.
    </DialogDescription>
  </DialogHeader>
  <DialogFooter>
    <Button onClick={() => setOpen(false)}>OK</Button>
  </DialogFooter>
</Dialog>
```

### Non-modal Dialog

```tsx
<Dialog open={open} onOpenChange={setOpen} modal={false}>
  <DialogHeader>
    <DialogTitle>Note</DialogTitle>
    <DialogDescription>
      This dialog does not block interaction with the rest of the page.
    </DialogDescription>
  </DialogHeader>
</Dialog>
```

### Sub-components

| Component | Description |
|---|---|
| `DialogHeader` | Container for the title and description. |
| `DialogTitle` | The heading of the dialog. Renders an `<h2>`. |
| `DialogDescription` | Supplementary text below the title. Renders a `<p>`. |
| `DialogFooter` | Container for action buttons, aligned to the end. |
