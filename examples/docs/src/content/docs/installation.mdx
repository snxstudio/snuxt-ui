## React

### 1. Set up Tailwind CSS v4

```bash
npm install tailwindcss @tailwindcss/vite
```

In your Vite config:

```js
import tailwindcss from '@tailwindcss/vite'

export default {
  plugins: [tailwindcss()],
}
```

### 2. Initialize snuxt

```bash
npx snuxt init
```

### 3. Add components

```bash
npx snuxt add button card dialog
```

### 4. Import CSS

In your main CSS file:

```css
@import 'tailwindcss';
@import './snuxt/css/base.css';
@import './snuxt/css/components/button.css';
@import './snuxt/css/components/card.css';
@import './snuxt/css/components/dialog.css';
```

### 5. Use components

```tsx
import { Button } from './snuxt/react/button'
import { Dialog, DialogHeader, DialogTitle } from './snuxt/react/dialog'
```

## Angular

### 1. Set up Tailwind CSS v4

```bash
npm install tailwindcss
```

Add to your `angular.json` styles array:

```json
"styles": ["src/styles.css"]
```

### 2. Initialize and add components

```bash
npx snuxt init --framework angular
npx snuxt add button card
```

### 3. Use standalone components

```typescript
import { ButtonComponent } from './snuxt/angular/button.component'

@Component({
  standalone: true,
  imports: [ButtonComponent],
  template: `<snx-button variant="primary">Click</snx-button>`,
})
export class AppComponent {}
```

## Manual Installation

If you prefer not to use the CLI, you can copy files directly from the [snuxt repository](https://github.com/thepsygeek/snuxt):

1. Copy `packages/css/base.css` to your project
2. Copy the component CSS from `packages/css/components/`
3. Copy the React/Angular component from `packages/react/src/` or `packages/angular/src/`
4. For interactive components, also copy the core logic from `packages/core/src/`
