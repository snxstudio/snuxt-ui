## Prerequisites

- **Node.js** 20+
- **pnpm**, **npm**, or **yarn**
- **Tailwind CSS v4** configured in your project

## Quick Start

### 1. Initialize snuxt in your project

```bash
npx snuxt init
```

This creates a `snuxt.json` config file and sets up the required directory structure.

### 2. Add components

```bash
npx snuxt add button
npx snuxt add card input dialog
```

Components are copied directly into your project — you own the source code.

### 3. Import the CSS

In your main CSS file, import the base tokens and component styles:

```css
@import 'tailwindcss';
@import './snuxt/css/base.css';
@import './snuxt/css/components/button.css';
@import './snuxt/css/components/card.css';
```

### 4. Use in your code

```tsx
import { Button } from './snuxt/react/button'

function App() {
  return <Button variant="primary">Click me</Button>
}
```

## What gets copied?

When you run `npx snuxt add button`, the CLI copies:

| File | Purpose |
|------|---------|
| `css/components/button.css` | Tailwind v4 styles |
| `react/button.tsx` | React component |
| `core/button.ts` | Core logic (interactive components only) |
