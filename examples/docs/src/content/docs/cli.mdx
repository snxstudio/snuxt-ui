## Commands

### `init`

Initialize snuxt in your project.

```bash
npx snuxt init
```

Creates a `snuxt.json` configuration file and sets up the output directory structure.

### `add`

Add one or more components to your project.

```bash
npx snuxt add <component...>
```

Examples:

```bash
npx snuxt add button
npx snuxt add card input textarea
npx snuxt add dialog --framework react
```

The CLI automatically resolves internal dependencies. For example, `npx snuxt add dialog` also copies `button` since Dialog depends on it.
